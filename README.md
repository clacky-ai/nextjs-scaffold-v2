# 投票系统

一个专业的项目评选投票平台，支持实名制注册、项目提交、多维度评分投票和结果统计。

## 🎯 核心功能

### 1. 用户管理
- ✅ **实名制注册与身份验证**：用户需要提供真实姓名、邮箱等信息
- ✅ **角色区分**：自动区分项目提交者与投票者
- ✅ **个人资料管理**：支持姓名、联系方式、所属团队等信息管理

### 2. 项目管理
- ✅ **项目提交功能**：支持项目标题、描述、团队成员、演示链接等完整信息
- ✅ **项目分类/标签**：可按技术领域、应用场景等进行分类
- ✅ **项目展示页面**：包含完整项目信息和评分情况的详情页

### 3. 投票机制
- ✅ **3票限制**：每用户最多可投3票，系统自动验证
- ✅ **防自投机制**：系统自动识别项目成员，禁止为自己的项目投票
- ✅ **多维度评分**：支持5个维度的单独打分（创新性、技术实现、实用性、展示效果、团队协作）
- ✅ **强制评价**：投票时必须提供文字评价和理由

### 4. 结果统计与展示
- ✅ **实时票数统计**：实时显示投票数量和排名
- ✅ **多维度分析**：各评分维度的详细分析和图表展示
- ✅ **评价汇总**：所有投票评价的集中展示
- ✅ **排行榜功能**：按平均分和投票数排序的完整排行榜

## 🏗️ 技术架构

### 前端技术栈
- **React 18 + TypeScript**：现代化前端框架
- **Vite**：快速构建工具
- **shadcn/ui + Radix UI**：高质量组件库
- **Tailwind CSS**：实用优先的CSS框架
- **TanStack Query**：服务器状态管理
- **Wouter**：轻量级路由
- **React Hook Form + Zod**：表单管理和验证

### 后端技术栈
- **Node.js + Express.js**：RESTful API服务器
- **TypeScript**：全栈类型安全
- **Drizzle ORM**：类型安全的数据库ORM
- **PostgreSQL**：主数据库
- **bcrypt**：密码加密
- **jsonwebtoken**：JWT认证

### 数据库设计
- **users**：用户信息表（实名制）
- **categories**：项目分类表
- **projects**：项目信息表
- **votes**：投票记录表（包含5维度评分）
- **user_vote_stats**：用户投票统计表
- **project_files**：项目文件表

## 🚀 快速开始

### 环境要求
- Node.js 18+
- PostgreSQL 数据库
- npm 或 yarn

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd voting-system
```

2. **安装依赖**
```bash
npm install
```

3. **配置环境变量**
```bash
cp .env.example .env
```
编辑 `.env` 文件，配置数据库连接等信息：
```
DATABASE_URL=postgresql://username:password@localhost:5432/voting_system
JWT_SECRET=your-secret-key
```

4. **初始化数据库**
```bash
npm run db:push
```

5. **启动开发服务器**
```bash
npm run dev
```

6. **访问应用**
打开浏览器访问 `http://localhost:3000`

## 📱 使用指南

### 用户注册和登录
1. 访问首页，点击"注册"标签
2. 填写真实姓名、邮箱、密码等信息
3. 注册成功后自动登录

### 提交项目
1. 登录后点击"提交项目"按钮
2. 填写项目标题、描述、分类等信息
3. 添加团队成员和项目链接
4. 提交后项目将出现在项目列表中

### 投票评分
1. 在项目列表中点击"投票"按钮
2. 对项目的5个维度进行评分（1-10分）
   - 创新性：项目的创新程度和独特性
   - 技术实现：技术方案的合理性和实现质量
   - 实用性：项目的实际应用价值
   - 展示效果：项目展示的完整性和清晰度
   - 团队协作：团队合作和项目管理能力
3. 填写详细的评价理由（必填）
4. 提交投票

### 查看结果
1. 点击"排行榜"查看所有项目排名
2. 点击项目"查看详情"查看具体评分和评价
3. 管理员可以访问"/admin"查看管理后台
